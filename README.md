# Telegram UserBot

Многофункциональный юзербот на Telethon с RP-командами, загрузкой медиа (YouTube / Instagram / TikTok) и стелс-функциями.

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

### Переменные окружения

| Переменная | Описание |
|-----------|----------|
| `API_ID` | API ID из my.telegram.org (обязательно) |
| `API_HASH` | API Hash оттуда же (обязательно) |
| `OWNER_ID` | Telegram ID владельца (обязательно) |
| `STRING_SESSION` | Строка сессии Telethon (опционально) |
| `PORT` | Порт Flask-статуса (по умолч. 8080) |
| `MEDIA_DIR` | Директория для скачанных файлов (по умолч. ./media) |

### RP-команды

Работают **только в личных сообщениях**. Для отправки RP-действия:

- **Любой пользователь** в ЛС — пишет команду, бот отвечает
- **Owner/Sudo** — реплай на чьё-то сообщение с командой (RP от имени бота к этому человеку)

Более 90 RP-действий. Список: `!rphelp`

## Зависимости

- `telethon` — работа с Telegram API
- `yt-dlp` — загрузка YouTube
- `requests` / `aiohttp` — HTTP-запросы
- `flask` — веб-статус
